# ML Course Labs

This repository contains lab assignments for the Machine Learning course (3rd year).

## Structure
- `laba_1/` — Lab 1 materials
- `laba_2/` — Lab 2 materials
- `laba_3/` — Lab 3 materials

Each lab folder contains:
- `All_TASK.md` — List of tasks for the lab
- `notebook_X.ipynb` — Jupyter notebook with solutions

## How to use
1. Open the desired lab folder.
2. Read the tasks in `All_TASK.md`.
3. Work with the corresponding notebook.
